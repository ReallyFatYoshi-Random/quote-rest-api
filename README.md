# Quotes Rest Api

> NOTE:
> You need NodeJS version 1.16.X or higher to run this application.

## Getting started
Open your terminal.

### Clone Repository
```sh
git clone https://github.com/ReallyFatYoshi-Random/quotes-rest-api.git
```

### Open Repository
```sh
cd quotes-rest-api
```

### Installation Dependancies
```sh
npm install
```

### Run the application
```sh
npm run start
```
